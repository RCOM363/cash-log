
# CashLog

CashLog is a full-stack web application designed to help users track cash transactions. This website allows users to manually log and manage expenses and incomes, offering a straightforward way to keep track of cash flow.




## Features

- **Logging Transactions:** Add, edit, and delete offline transactions.
- **Categorization:** Assign each expense/incomes to a single custom category.
- **Search & Filter:** Filter transactions by category, search by title/description.
- **Data Visualization:** View detailed insights and trends through charts.
- **Export Transactions:** Export your data as an Excel sheet for easy record-keeping.
- **Responsive Design:** Responsive design, optimized for desktop and mobile.
